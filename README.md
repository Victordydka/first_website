📌 Online Store Website (PHP + MySQL + Docker)

Веб-приложение интернет-магазина с базовой системой регистрации и авторизации пользователей, построенное на PHP и MySQL с использованием Docker.

Возможности:
🧑 Регистрация пользователей

🔐 Авторизация пользователей

🔑 Хеширование паролей (password_hash)

💾 Хранение данных в MySQL

🌐 Веб-интерфейс (HTML/CSS/JS)

🐳 Запуск через Docker


Проект демонстрирует:
- базовую backend-разработку на PHP
- работу с базой данных (MySQL)
- обработку форм и HTTP-запросов
- контейнеризацию приложения (Docker)
- построение простой клиент-серверной архитектуры


🛠 Стек технологий
Backend:
- PHP 8.3
- mysqli
- Database
- MySQL 8.0

Frontend:
- HTML
- CSS
- JavaScript

DevOps:
- Docker
- Docker Compose
- phpMyAdmin
